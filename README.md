# Pathfinder Visualiser

- Created an application for visualising path-finding algorithms.
- Used pure typescript without any UI library.
- Made use of pub-sub architecture to handle complex events and manage global state.
- Implemented different pathfinding algorithms with the ability to create obstacles.
